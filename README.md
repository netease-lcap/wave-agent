# lcap-code

一个基于 Ink 和 React 的终端编程代理。

## 使用方法

### 命令行使用

```bash
# 打开当前目录
npx lcap-code

npx lcap-code --help
```

- [MCP协议](docs/MCP.md) - Model Context Protocol 支持
- [分页功能](docs/PAGINATION.md) - 文件列表分页实现

- 所有日志都写入到文件中，不输出到控制台
