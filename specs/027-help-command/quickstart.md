# Quickstart: Help Command

## How to use

1. Start the Wave Agent CLI.
2. In the chat input, type `/help` and press `Enter`.
3. The Help View will appear, overlaying the chat interface.

## Navigation

- **Switch Tabs**: Press `Tab` to cycle between "General", "Commands", and "Custom Commands".
- **Navigate Commands**: In the "Commands" or "Custom Commands" tabs, use the `Up` and `Down` arrow keys to select a command.
- **View Description**: The description for the currently selected command is displayed automatically.
- **Close Help**: Press `Esc` to close the help view and return to the chat.

## Key Bindings (General Tab)

The "General" tab lists important keyboard shortcuts:
- `@`: Reference files
- `/`: Commands
- `!`: Shell commands
- `Ctrl+R`: Search history
- `Ctrl+O`: Expand/collapse messages
- `Ctrl+T`: Toggle task list
- `Ctrl+B`: Background current task
- `Ctrl+V`: Paste image
- `Ctrl+J`: Newline
- `Shift+Tab`: Cycle permission mode
- `Esc`: Interrupt AI or command / Cancel selector / Close help
