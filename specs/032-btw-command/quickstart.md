# BTW Command Quickstart

The `/btw <question>` command allows users to ask side questions to the AI without using tools.

## Usage

1. Type `/btw <question>` in the chat input box.
2. Press Enter.
3. The side question and its answer will be displayed at the bottom of the chat interface.
4. Press **ESC** to dismiss the side question view and re-enable the main input box.

## Example

```bash
/btw what is the current time?
```

The AI will respond with the current time without using any tools.
